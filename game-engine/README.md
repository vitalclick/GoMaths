# game-engine

Unity-based RPG and gamification engine.

## Status
**Phase 4. Not in MVP scope.**

This directory is a placeholder. The strategy doc treats the RPG layer as a later phase because it is effectively a separate product (game designers, animators, sound, multiplayer netcode) and should not block MVP delivery.

When Phase 4 begins:
- Stack: Unity LTS
- Multiplayer: Firebase real-time backend (per strategy doc)
- Integration: deep-linked from the React Native shell so the RPG is a "world" inside the existing app rather than a separate install
