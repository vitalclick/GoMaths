# admin-portal

Next.js web application — internal admin and curriculum management.

## Scope
- **MVP:** Minimal — read-only learner activity view for the pilot team
- **Phase 2:** Curriculum Management Portal (authoring without git), school/teacher admin, parent dashboards

## Stack
- Next.js (App Router) + TypeScript
- Tailwind CSS
- TanStack Table
- Recharts
- Auth0 or Firebase Auth (decision deferred)

## Status
Not yet scaffolded. Initialise with `npx create-next-app@latest admin-portal --typescript --tailwind --app` after backend-api auth is in place.
