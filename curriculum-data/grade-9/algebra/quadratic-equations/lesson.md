# Solving Quadratic Equations by Factorising

A **quadratic equation** has a squared term, for example $x^2 - 5x + 6 = 0$.
Unlike a linear equation, it usually has **two** solutions.

## The zero-product principle

If two things multiply to give zero, then **at least one of them must be
zero**:

$$\text{if } A \times B = 0 \text{ then } A = 0 \text{ or } B = 0$$

This is the key idea. To use it we first make one side equal to zero, then
factorise.

## The method

1. Rearrange so the equation reads $\dots = 0$.
2. **Factorise** the left-hand side.
3. Set **each factor** equal to zero.
4. Solve each small equation.

**Example:** solve $x^2 - 5x + 6 = 0$.

Factorise: $(x - 2)(x - 3) = 0$.

So $x - 2 = 0$ or $x - 3 = 0$, giving $x = 2$ or $x = 3$.

## Watch out for two traps

- **Never divide both sides by $x$.** In $x^2 - 4x = 0$ that would throw
  away the solution $x = 0$. Factorise instead: $x(x - 4) = 0$.
- **A square has two roots.** $x^2 = 25$ gives $x = 5$ **or** $x = -5$, not
  just $x = 5$.

## Always check

Substitute each solution back into the original equation to confirm it
gives a true statement.
