# The Mean of a Data Set

The **mean** (everyday "average") tells you the typical value in a set of
numbers.

$$\text{mean} = \frac{\text{sum of all values}}{\text{number of values}}$$

## Worked examples

Two values: $\dfrac{12 + 18}{2} = \dfrac{30}{2} = 15$.

Three values: $\dfrac{10 + 20 + 30}{3} = \dfrac{60}{3} = 20$.

## The mean doesn't have to be a whole number

$$\frac{3 + 7 + 11 + 9}{4} = \frac{30}{4} = 7.5$$

Leave it as a decimal (or fraction) — don't round unless the question
asks you to.

## Two-step method

1. **Add** every value to get the total.
2. **Divide** the total by **how many** values there are.

> The most common slip is dividing by the wrong count — always recount how
> many numbers are in the set.
