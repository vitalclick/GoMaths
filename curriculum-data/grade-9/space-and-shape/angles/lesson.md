# Angle Relationships

Many geometry problems come down to a few facts about how angles add up.

## Angles on a straight line

Angles on a straight line add to **180°**.

$$180° - 130° = 50°$$

## Complementary and supplementary

- **Complementary** angles add to **90°**: $90° - 35° = 55°$.
- **Supplementary** angles add to **180°** (same as a straight line).

## Angles in a triangle

The three interior angles of any triangle add to **180°**.

$$180° - 65° - 40° = 75°$$

In a right-angled triangle, one angle is already $90°$, so the other two
add to $90°$: $180° - 90° - 47° = 43°$.

## Angles in a quadrilateral

The four interior angles add to **360°**.

$$360° - 90° - 90° - 100° = 80°$$

## Quick reference

| Shape / situation     | Angles sum to |
| --------------------- | ------------- |
| Angles on a line      | 180°          |
| Complementary pair    | 90°           |
| Triangle              | 180°          |
| Quadrilateral         | 360°          |
